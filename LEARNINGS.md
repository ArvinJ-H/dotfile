# Learnings

Preferences, working-style observations, and domain knowledge. Read on session start.
Confirmed across 2+ sessions -> propose CLAUDE.md amendment via the Improve workflow (/meta).
Graduated entries move to `LEARNINGS-ARCHIVE.md`.

### Entry format

```
### [short title] -- YYYY-MM-DD -- scope:<scope>
**Observation**: what was learned
**Confidence**: low | medium | high
**Sessions**: [YYYY-MM-DD, YYYY-MM-DD]
**Tags**: free-form, comma-separated
```

**Scope** (where the code lives, not where you're working from):
- `global` -> `~/.claude/CLAUDE.md` or global skills/hooks
- `<project-name>` -> project-level CLAUDE.md or skills

**Sessions**: date-based list; the Improve workflow counts distinct sessions automatically.
**Tags**: optional, free-form. Used by the Improve workflow for cross-category pattern detection.
